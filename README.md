# Spotify Clone

This project is a clone of the Spotify web application. It includes features such as browsing playlists, creating new playlists, and following podcasts.

## Features

- Browse your library
- Create new playlists
- Follow podcasts

## Technologies Used

- HTML
- CSS
- JavaScript
